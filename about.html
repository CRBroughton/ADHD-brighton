Bootstrap

Bootstrap is one of the most popular front-end frameworks for fast and sleek websites. Bootstrap helps enable developers to more easily create responsive web design with its flexbox grid system.

jQuery

jQuery is a Javascript library, allowing developers access to a feature-rich tool set. jQuery is used on this site for button click detection, ensuring users are brought to the correct part of each page.

Linear Gradient

The hero box (beneath the navigation bar) features a transparent linear gradient that subtly dulls the background image, giving it a more subdued appearance.

Google Maps API

Using the Google Maps API, those interested in this sites services can more easily find the locations and its services using the featured map.
